## Functionality
The group chose to implement set 2.

### Left Bit Shift
```
print 5 << 2;
print 1 << 5;
```

### Direct Length Comparison for Strings
```
print “hello” == 5;
print “Hello” < 5;
print “Nice” >= 4;
print 5 >= “Nice”;
```

### Support Plus sign before a number.
```
var x = +123;
print x;
print +123;
var y = 5 + +123;
print y;

var x = ++123;
```

### String multiplication
```
print "Hello" * 3;
print "beep boop " * 2;
print 3 * "Hello"; 
```
